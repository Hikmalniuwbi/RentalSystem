*   **Visual:** Kursor menunjuk ke Statistik (Booking, Aktif) dan tabel stok real-time.

## 3. Manajemen Inventaris & Harga Paket (0:45 - 1:15)
*   **Visual:** Pindah ke menu **Inventaris**, buka dialog **Kelola Harga**.

Contohnya, harga per hari akan otomatis menyesuaikan menjadi lebih murah jika pelanggan mengambil paket sewa 3 hari atau satu minggu."

## 4. Sistem Transaksi & Validasi Stok (1:15 - 1:45)
*   **Visual:** Pindah ke menu **Transaksi**, pilih tanggal, dan coba masukkan barang ke keranjang.

Saat Admin memilih tanggal sewa, sistem secara otomatis mengecek ketersediaan stok di database pada periode tersebut. Jika stok habis, admin tidak bisa menambahkannya ke keranjang, sehingga kesalahan fatal dalam pemesanan dapat dihindari sepenuhnya."
